# Выгрузка ведомости (Excel / PDF)

Экран **«📊 Ведомость»** → кнопки **«Excel»** и **«PDF»** выгружают ведомость
успеваемости выбранного предмета в файл (`vedomost_<Предмет>.xlsx` / `.pdf`).

Логика формирования вынесена в `report_export.py` (без БД и Kivy → headless-тесты
в `tests/test_report_export.py`). На вход подаётся готовая ведомость из
`database.subject_gradebook()` + сводка `database.subject_summary()`.

## Excel (`openpyxl`) — основной формат
- Чисто-Python библиотека, кириллица без проблем.
- Содержит: шапку (предмет, группа, план/проведено/осталось, средний балл по
  предмету) и таблицу «ФИО / средний балл / кол-во оценок / присутствовал /
  отсутствовал».

## PDF (`reportlab`)
- У встроенных шрифтов reportlab нет кириллицы, поэтому `report_export` пытается
  зарегистрировать системный TTF (DejaVuSans на Linux/CI, Arial на Windows,
  Arial на macOS). Если ни один не найден — PDF всё равно создастся, но кириллица
  может отображаться некорректно; в этом случае используйте Excel.

## Зависимости
```
pip install openpyxl      # Excel — рекомендуется добавить и в сборку (pure-python)
pip install reportlab     # PDF — опционально
```
Обе библиотеки импортируются **лениво** внутри `report_export`; при их отсутствии
кнопка выгрузки покажет понятное сообщение, а приложение не упадёт.
